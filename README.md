## Payment Flow

1. Click the Payment button.
2. The Razorpay Test Mode payment window will open.
3. Select any available payment option (UPI, Card, Net Banking, etc.) and complete the test payment.
4. If the payment is successful, a Payment Success confirmation page will be displayed.
5. After a few seconds, the user is automatically redirected back to the  Home page.
6. If the payment fails, the Payment Failed page is displayed, allowing the user to try again.
